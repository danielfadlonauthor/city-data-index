# City Data Index
A working reference list of open datasets useful for researching infrastructure and urban governance in mid-size cities.
Maintained by Daniel Fadlon alongside research for Infrastructure & Cities (danielfadlon.substack.com).
## Included datasets
- Transit ridership: GTFS feeds for 40+ cities
- Water infrastructure: WHO/UN-Habitat open data
- Municipal budget data: OpenSpending
- Population density grids: GHS-POP (EU JRC)
